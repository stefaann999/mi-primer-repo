# Mi Primer Repo

Este es un proyecto de práctica para aprender a usar Git, GitHub y Pull Requests.

## Qué hace este proyecto

Nada todavía. Es solo un punto de partida para aprender el flujo de trabajo:

1. Clonar o crear un repositorio
2. Crear una rama
3. Hacer un cambio
4. Abrir un Pull Request
5. Revisar y fusionar (merge)
